# How to Add Images

## Divyansh Gupta's Photo

To add Divyansh Gupta's image to the website:

1. **Save the image file** as: `team-divyansh-gupta.jpg`
2. **Place it in**: `assets/images/` folder
3. **Full path**: `assets/images/team-divyansh-gupta.jpg`

The image will automatically appear on the **Our Team** page (`team.html`).

### Image Specifications (Recommended)
- **Format**: JPG, PNG, or WebP
- **Size**: At least 400x400 pixels (square format works best)
- **Aspect Ratio**: 1:1 (square) is ideal for the circular display
- **File Size**: Optimize to under 500KB for faster loading
